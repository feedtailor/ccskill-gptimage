## タスク概要

`generated_images/` 配下の画像とメタデータ JSON サイドカーを一覧/閲覧する Web ギャラリーを追加する。`ccskill-gptimage gallery` で起動。

ccskill-nanobanana 側に同様の Web ギャラリー設計タスク(issue 007)があるため、**両スキル共通の gallery を別パッケージ**(例: `ccskill-image-gallery`)として切り出し、両方から呼べるようにする方針も検討する。

## 設計項目

### 技術スタック候補
- サーバ: Python `http.server` または最小 Flask
- フロント: 静的 HTML/CSS/JS(LP と同じく素朴に)
- 依存最小化を重視

### 機能
- 画像一覧(サムネイルグリッド + メタデータ)
- プロンプト全文表示 + コピー
- フィルタ(モデル / 日付 / キーワード)
- 拡大表示
- ソート(日付 / quality / size)
- メタデータ JSON のダウンロード

### 共通化検討
- nanobanana の `generated_images/` フォーマットと統合できるよう、メタデータスキーマを擦り合わせる
- ggcskill-image-gallery として独立リポにする vs 各スキルに同梱するか決定

## 完了条件

- [ ] `ccskill-gptimage gallery` でブラウザが開き、画像一覧が表示される
- [ ] 共通化方針(同梱 / 別パッケージ)が決定し、ADR に記録
